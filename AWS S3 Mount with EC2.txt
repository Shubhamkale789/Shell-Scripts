1. Create a EC2 instance.
2. Create an S3 bucket.
3. Create a role for EC2 with S3 in IAM.
4. Install AWS Cli on EC2 amazon linux instance.
   curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
   unzip awscliv2.zip
   sudo ./aws/install
6. Install S3 mount package on EC2.
   sudo apt install s3fs
7. go to home directory and create a new directory.
8. Mount S3 bucket with the created directory
   s3fs (bucket name) (directory name)
9. Go to the created directory and we can view all the files uploaded inside bucket.
   Now we can create and view files inside the mounted bucket. 
10.We can unmount the s3 bucket from the directory by running the following command.
   umount (/home/ubuntu/bucket)
